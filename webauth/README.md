上网身份认证程序

通过发送 POST 请求提交用户名和密码到网络认证服务器完成认证请求。


使用说明：
1、windows xp/vister/win7 需要安装python 解释器<http://www.python.org/getit/>
2、linux /osx 一般自带pyton解释器，可以直接允许
3、android 安装 SL4A 和 PythonForAndroid <https://code.google.com/p/android-scripting/> 
4、windos 下可以将该文件放到启动目录（C:\Documents and Settings\***\「开始」菜单\程序\启动\）
开机自动完成上网认证。
